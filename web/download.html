Download the Project


There are two ways to download the project:
  1. By svn:

    Just go to the place you want to put the files and run the following command in the shell:
    svn checkout http://liquid-crystal-simulation.googlecode.com/svn/trunk liquid-crystal-simulation-read-only

    The project will be in the directory: ./liquid-crystal-simulation-read-only


  2. Download directly:

    The project source code: lc_simulation_src.tar.gz The project web code and simulations: lc_simulation_web.tar.gz
    To unpack the source files, put the downloaded file in the directory where you want the project to be, and run the following command in the shell:

    tar -xzvf lc_simulation_src.tar.gz

    All of the project's sources and run scripts will be in the directory: ./lc_simulation_src

    The second file contains the web content and the simulation data and statistics that are presented on this website. To unpack those files, put the downloaded file in the same directory as the previous one, and run the following command in the shell:
    tar -xzvf lc_simulation_web.tar.gz

    All of the projects web content and simulation run data will be added to the directory: ./lc_simulation_src If you performed this action in the same directory as the previous command, the added directories will be "web" and "runs" within the same project directory.


Download Additional Programs


The following programs may be needed in addition to the project files:
  1. python 2.4 (and any version above that):
    Needed in case it is not installed - it should be available on any linux system.


  2. [OPTIONAL] aviz 5.6 (and any version above that):
    Needed in order to render and display the model images. The simulator can still run and generate statistics, graphs of Energy and Temperature, and a timeline, without using AViz.


  3. [OPTIONAL] swig 1.3 (and any version above that):
    Needed if you want to be able to use the fast implementation of the Gay-Bernes potential (implemented in C++). It is used to wrap the C++ code into Python that can be used by the simulator.
