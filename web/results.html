2D Simulation - Conclusions

Most of the analysis was done using the statistics page (Explanations for how to use it can be found here).

First of all, in both simulations the phase changes are visible to some extent.
In the slow cooling process more phase changes can be seen, and they are more prominent as well.
The fast quench happens in both cases, and is an indication that even the slow cooling isn't slow enough, and solidification occurs at the end.

Secondly, when viewing the Energy vs. Temperature graph for both processes, it can be seen that they overlap in most places, and the general trend is the same.
While the faster cooling is more erratic, both systems still follow the same path, and both start loosing energy rapidly at almost the same point.
This can be seen clearly in the graph itself:

Finally, from viewing the timeline of both systems it can be seen that we reach a lower energy at the final temperature in the slower cooling system.
This implies that, as in real life physics, that with slower cooling and carefull equalibiration of the system at each temperature, we can achieve a lower total system energy.
This of course makes sense both physically and in the Monte Carlo simulation itself, and can be seen in the following graph:

Most phase changes are visible in these two simulations, and the Monte Carlo method demonstrates most of the physics involved.
This is especially evident in the changes noted above between slow and fast cooling and their effects on the system.

3D Simulation - Conclusions

Most of the analysis was done using the statistics page (Explanations for how to use it can be found here).

We will discuss several points we arrived at:
  • As with the 2D simulations, we witnessed several differences between the fast and slow cooling.
  • We also noticed a difference in the slow cooling here, compared to the 2D slow cooling.
  • Finally, we will compare as a whole the 2D system to the 3D system.

    • To the first point, in both 3D simulations the phase changes are visible to some extent.
    As in the 2D simulation, in the slow cooling process more phase changes can be seen, and they are again more prominent as well.
    This is good, since it shows consistency in the results of the simulations, and that different models behave the same in the same physical conditions.
    The fast quench happens in both cases (as in the 2D variant), and once again shows that even the slow cooling isn't slow enough, and solidification occurs at the end.

    When we view the Energy vs. Temperature graph for both processes, it can be seen that they overlap almost exactly, and the general trend is the same (as we have seen in the 2D simulations).
    This again shows consistency in the results, in both 2D and 3D.
    The faster cooling is as expected more erratic, and they both converge in the same area of temperatures and start loosing energy rapidly.
    This can be seen clearly in the graph itself:

    When viewing the timeline of both systems it can be seen that we again reach a lower energy at the final temperature in the slower cooling system.
    This is very encouragin evidence, as it shows that twice now, in two different systems, with slower cooling and carefull equalibiration of the system at each temperature, we can achieve a lower total system energy.
    This of course makes sense both physically and in the Monte Carlo simulation itself, and can be seen in the following graph:

    To the second point, the slow cooling in the 3D simulation is decidedly different in several aspects from its 2D counterpart.
    For one thing, it is much harder to notice the subtle changes in phase in the 3D model than in the 2D model, and not only because of the visual representation.
    In both runs, because we were somewhat limited by computer time and resources (we did not want to run a simulation that took weeks), we had to limit the number of Metropolis steps to 10,000.
    This is not a high number, and it would be much better had we used between 40,000 and 60,000 steps, but that would take much much longer than the few days it took to run our simulations.
    While in the 2D model this was apparently enough to see most of the results, in the 3D model it was simply not enough to get very good results.
    A second run of this slow cooling process with more Metropolis steps would allow the system to reach a more stable equalibrium at each temperature, and allow for more minute details to show.
    Note that the system is fully capable of running said simulations, and much more intensive runs were made on smaller systems.

    The final point in the results regards comparing the 2D simulation as a whole to the 3D simulation.
    It is interesting to note that in essence these two systems are not all that different.
    The simulation behaves the same, and while the energy scale is very different, and the temperatures at which phase changes occur are somewhat different, the general route both systems take is the same.
    Of course this generalization cannot be made based on the few simulations we ran, but it was mainly reasurring to us that the physics stays the same in different system parameters, and the simulation system itself is consistent.
